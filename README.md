# 호텔 예약 서비스 프론트엔드 레포지토리  
Node.js 백엔드 및 MySQL 데이터베이스와 연동

## 배포
- **Frontend (React)**: Cloudtype
- **Backend (Node.js)**: https://github.com/kennywestt/nodejs_hotel (Cloudtype)
- **Database (MySQL)**: https://github.com/kennywestt/nodejs_hotel (Cloudtype)
- API 연동을 통해 로그인/회원가입/예약 기능 제공

## Tech Stack
- React
- React Router
- Axios
